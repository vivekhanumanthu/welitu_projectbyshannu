# We Lit You - Luxury Showcase Website

Portfolio-first luxury wedding planning website for **We Lit You**.

## Stack

- Next.js
- React
- TailwindCSS
- Framer Motion
- Vercel-ready deployment

No Docker, PostgreSQL, or backend is required for this stage.

## Pages

- `/` Home
- `/about`
- `/services`
- `/portfolio`
- `/gallery`
- `/contact`

## Features

- Cinematic hero section
- Luxury visual design system (champagne / beige / ivory / black)
- Masonry gallery with lightbox and filters
- Portfolio wedding showcase cards
- WhatsApp floating inquiry button
- SEO setup: metadata, schema, sitemap, robots

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000`

## Deploy (Vercel)

1. Import the `frontend` folder into Vercel.
2. Deploy with default Next.js settings.

## Brand Assets

- Replace `frontend/public/logo.svg` with your final uploaded logo asset.
- Replace placeholder images in `frontend/public/images/` with real wedding visuals.
